# Dystopia Launcher

Canal public de telechargement et de mises a jour du launcher du serveur DayZ roleplay **Dystopia**.

Le code source est prive. Ce depot ne contient que les versions publiees (Releases) et le manifeste de mise a jour `latest.json` que le launcher consulte automatiquement.

## Telechargement

Les versions sont publiees dans les **Releases** de ce depot. Telecharger le fichier `.msi` de la derniere version, puis l'installer.

Au premier lancement, Windows peut afficher un avertissement « editeur inconnu » (le launcher n'est pas encore signe) : cliquer sur « Informations complementaires » puis « Executer quand meme ». Des sommes de controle SHA-256 sont jointes a chaque version pour verification.

_(Section a completer a la premiere publication.)_

## Mises a jour

Une fois installe, le launcher se met a jour tout seul depuis ce canal, en verifiant `latest.json` au demarrage. Rien a retelecharger a la main.
